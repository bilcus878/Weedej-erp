Ahoj! Tohle je kompletní účetní program napojený na SumUp API.

📚 ZAČNI TADY:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Otevři soubor: START_HERE.md
   → Přehled projektu a co číst dál

2. Pro instalaci:
   → QUICK_START.md (rychlý) nebo
   → INSTALACE_KROK_ZA_KROKEM.md (detailní)


🚀 RYCHLÁ INSTALACE (5 minut):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Nainstaluj PostgreSQL:

   Docker (doporučeno):
   docker run --name ucetni-db -e POSTGRES_PASSWORD=heslo123 -p 5432:5432 -d postgres:14

   Nebo stáhni z: https://www.postgresql.org/download/

2. Uprav .env soubor:
   DATABASE_URL="postgresql://postgres:TVOJEHESLO@localhost:5432/ucetni_db"
   (Změň TVOJEHESLO)

3. Spusť instalaci:
   Klikni na: scripts\setup.bat

4. Spusť aplikaci:
   Klikni na: scripts\start.bat

5. Otevři prohlížeč:
   http://localhost:3000
